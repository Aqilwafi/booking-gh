# Guesthouse Booking App

A modern guesthouse booking web application built with **Next.js**, **Supabase**, and **TailwindCSS**.  
This app allows users to browse rooms, book their stay, and lets admins manage rooms and bookings easily.

---

## Features

- User registration & login (email/password or Google OAuth)
- Admin dashboard for managing rooms and bookings
- Room categories & gallery with image slider
- Booking system with status (pending, confirmed, canceled)
- Responsive design for mobile and desktop
- Integration with Supabase Auth and database

---

## Tech Stack

- Frontend: Next.js (App Router), TailwindCSS
- Backend: Next.js API Routes (not used yet)
- Database: Supabase Postgres, Supabase Storage (not used yet)

---

## Setup / Installation

1. Clone the repository:

```bash
git clone this-repo-url
cd booking-gh
```

2. Install dependencies:

```bash
npm install
```

3. Run development server:

```bash
npm run dev
```

4. Visit http://localhost:3000
